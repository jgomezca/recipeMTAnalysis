Setup for MT Analysis
=====================

Usage
----

Setup the relevant CMSSW version

    cmsrel CMSSW_5_3_2_patch4
    cd CMSSW_5_3_2_patch4/src
    cmsenv 

Clone this repo into your /src/ area

    git clone git://github.com/ttbarMET-at-IPHC/recipeMTAnalysis.git ./
    
Launch the install
    
    source install.sh
